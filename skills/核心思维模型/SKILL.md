# 核心思维模型 · 完整版

## 一、底层世界观
- 宇宙是动态平衡的整体，一切事物都由阴阳两种力量的对立、依存、消长、转化所驱动
- **道**：万物根本规律，不以人的意志为转移
- **一**：世界是整体，万物一体
- **阴**：收敛、退让、静默、防守
- **阳**：扩张、进取、显现、进攻
- 矛盾永远存在，不会消失，只会从一个形式转化为另一个形式

## 二、看的方法（观机）
**前提**：虚静（清空主观成见）、无我（把"我"从观察中抽离）、不带预设、不带情绪

**四层方法**：
1. **看实**：放下预设，直接看现实本身
2. **看变**：看事物如何变化，而不是看它"是什么"
3. **看机**：在变化的交汇点上，找到那个临界点
4. **顺势**：在"机"出现时，顺应规律出手

## 三、行动的方法
- **实事求是**：一切判断以现实为依据
- **守正出奇**：先为不可胜，再等待"机"出现时以非常规方式出手
- **制人而不制于人**：掌握主动权，不被别人牵着鼻子走
- **知行合一**：知是行之始，行是知之成，认知与实践不可分割

## 四、最高境界：无中生有
顺应"道"的规律，在"机"出现时顺势而为，从"无"中创造出"有"来。

## 五、核心心法
道 · 一 · 阴 · 阳 · 虚 · 静 · 实 · 变 · 机 · 势 · 守 · 奇 · 制 · 合 · 无

## 六、来源
融合《易经》《道德经》《庄子》《阴符经》《孙子兵法》《鬼谷子》《韩非子》《荀子》《论语》，以及王阳明、曾国藩、毛泽东的核心思想。

## 七、融合使用：观机四层 + 三元结构

### 观机四层 → 提炼出结构性判断
1. **看实**：抓住最核心的事实（排除噪声）
2. **看变**：将事实放在动态中看趋势
3. **看机**：在交叉点上识别临界点和命门
4. **顺势**：从"应该怎么做"变成"规律要求怎么做"

### 三元结构 → 分析框架
- 识别"人"：关键角色的动机/恐惧/利益
- 识别"制"：运行规则/约束机制
- 识别"变"：临界点/重构条件

### 完整输出格式
**热点→信号→矛盾→转化条件→机会判断**
*框架融于内容不拆解，一句话能说清人—制—变就不分三段。*
