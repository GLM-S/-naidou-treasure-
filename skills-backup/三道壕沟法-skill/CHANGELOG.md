## v1.0.0 — 初始发布

基于 taste-skill 的结构骨架，将静渊的三道壕沟法重新结构化。

### 核心架构
- **三组可调参数**：信息宽度 (1-10) / 深度层次 (1-10) / 风险保守度 (1-10)
- **五道壕沟**：第零道（值不值得干）→ 第一道（广撒网）→ 第二道（扎深根）→ 第三道（开口子）→ 第四道（交付物）
- **参数推算表**：问题信号 → 自动推算三组参数值
- **信道选择指南**：不同问题类型匹配不同搜索策略
- **反思维陷阱清单**：信息/逻辑/表达三类常见陷阱
- **验收清单**：交付前逐项自检
- **🔔 机会标记协议**：标记有深挖价值的选题

### 设计决策
- 使用 taste-skill 的「三旋钮 + 场景推算表」结构，适配认知筛选而非前端设计
- 增加「第零道」元壕沟——这是三道壕沟法区别于其他方法论的核心
- 输出模板借鉴 taste-skill 的 pre-flight check 理念，但做了中国语境适配
