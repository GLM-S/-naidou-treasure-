### 5. Python 批量替换留下双逗号

### 6. q1b 的 mp3 文件名写错

## 总结的规则

| 规则 | 具体做法 |
|------|---------|
| 函数签名+全量 grep | 改参数签名 → grep 所有调用点并更新 |
| 补丁阈值=3 | 同文件补丁超过3次 → 暂停评估是否要重写 |
| 复杂JS用edit工具 | 不用 sed 处理 JS 字符串 |
| 用户重复=关键信息 | 用户说同一句话超过2遍 → 记录到笔记，不是啰嗦 |
| 镜像修复 | 修复A页面bug → 顺手检查B、C页面是否有同类问题 |
| 交付前全流程测试 | 修改后必须走完整用户路径，不能只测改的那块 |
| 先加逗号后补值 | 在对象中插入新属性时，确保前后逗号完整 |
