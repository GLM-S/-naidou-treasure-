# 三道壕沟法 · 炒股决策版
## AI 可执行说明书 v0.1

---

## 一、技能概述

**用途：** 判断今天要不要开仓买某只股票。  
**适用对象：** A股短线/波段交易，非长持。  
**输出产物：** 一份 300 字核心矛盾笔记 + 三种情景预案 + 仓位建议。  
**不可输出场景：** ①情绪不稳定时 ②大盘暴跌时 ③不了解的股票。

---

## 二、执行前检查（第零道）

在执行以下任何步骤之前，先问四个问题。只要有一题答案是「否」，**立即停止，不执行后续**：

| 问题 | 含义 | 通过标准 |
|------|------|---------|
| 这只股票我了解吗？ | 不是听消息追的 | 能说出它的主营业务和近期逻辑 |
| 最近三天我看过它的K线吗？ | 不是闭眼买 | 至少看过日线和周线 |
| 我能接受它跌10%吗？ | 仓位控制 | 亏了不慌，睡得着觉 |
| 今天心情正常吗？ | 情绪检测 | 没有因为亏钱想翻本，也没有因为赚了想加仓 |

---

## 三、执行步骤

### 第一步：雷达扫描（收集原始信号）

同时查三个方向，每个方向控制在 3 分钟内：

**方向A - 技术面：**
```
1. 打开K线（日线）
2. 看最近5根K线的形态：
   - 是连续阳线/阴线/A字/横盘？
   - 有没有突破关键均线（MA5/MA10/MA20）？
3. 看成交量：
   - 放量上涨还是缩量上涨？
   - 放量下跌还是缩量下跌？
4. 记录一句话结论
```

**方向B - 资金面：**
```
1. 查Level-2资金流向
2. 看主力净流入/流出金额
3. 看散户跟风情况（有没有明显跟买）
4. 记录一句话结论
```

**方向C - 情绪面：**
```
1. 扫一眼龙虎榜（如果有）
2. 看今天有没有重大利好/利空公告
3. 看所属板块整体表现（涨/跌/轮动）
4. 记录一句话结论
```

---

### 第二步：跨源验证（打出结论）

把三个方向的结果放一起对比：

```
对比规则：
  - 三个方向一致 → 结论为「强烈」，直接下一步
  - 两个方向一致，一个相反 → 结论为「参考一致方向」，但仓位减半
  - 三个方向全部不一致 → 结论为「冲突」，标记为「不操作日」

输出要求：
  用一句话写清楚：
  「技术面显示___，资金面显示___，情绪面显示___，整体判断___。」
```

---

### 第三步：写 300 字核心矛盾笔记

**格式固定，不可省略：**

```
【核心矛盾】
（用一句话写出这只股票当前最大的矛盾。例如：「业绩好但主力在跑」）

【技术面证据】
（1-2句话，引用第一步的记录）

【资金面证据】
（1-2句话，引用第一步的记录）

【情绪面证据】
（1-2句话，引用第一步的记录）

【我的判断】
（1-2句话，我自己怎么想。区别于证据，这里是主观判断）
```

---

### 第四步：三种情景预案

针对第三步的判断，写出三种情况的应对预案：

```
【如果涨了：涨多少？怎么处理？】
  - 涨2%以内：持有不动
  - 涨2%-5%：减仓三分之一
  - 涨5%以上：减仓一半

【如果震荡：震荡多久？怎么处理？】
  - 震荡3天内：持有不动
  - 震荡超过一周：减仓一半，等方向

【如果跌了：止损点在哪？怎么处理？】
  - 跌到止损线（必须提前设定）：无条件卖出
  - 触及止损线前的预警位：减仓一半
```

---

### 第五步：仓位建议

```
根据第二步的结论输出：

【强烈做多】 → 建议仓位：总资金的 30%以内
【参考方向】 → 建议仓位：总资金的 15%以内
【冲突/不操作】 → 建议仓位：0%
```

---

## 四、验收标准

写完以上全部后，检查：

```
✅ 第零道四问都过了
✅ 三个方向都查了，不是只看一个
✅ 300字笔记写全了，没有跳步骤
✅ 止损位已经定好了
✅ 仓位建议没有超过总资金30%
✅ 情绪正常，没有追涨杀跌的冲动
```
