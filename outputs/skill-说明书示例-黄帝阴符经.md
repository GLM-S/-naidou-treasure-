# 《黄帝阴符经》知识提取说明书
## AI 可执行技能 v0.1

---

## 第一部分：元技能（通用骨架—所有书都能用）

**无论哪本书，提取知识都分这五步：**

| 步骤 | 做什么 | 输出 |
|------|--------|------|
| ① 通读 | AI通读全文，识别核心领域 | 一句话判断书的类型 |
| ② 定义实体 | 根据书的类型确定实体类别 | 实体类别清单 |
| ③ 提取实体 | 逐句标注 | 实体数据表 |
| ④ 提取关系 | 连接实体之间的关系 | 关系数据表 |
| ⑤ 结构化输出 | 按预定格式输出 | 结构化知识文档 |

## 第二部分：定制层（《黄帝阴符经》专用）

### 0. 书本信息

```
书名：黄帝阴符经（简称《阴符经》）
字数：约300字
类型：道家/内丹哲理经典
结构：三章（神仙抱一演道章 / 富国安民演法章 / 强兵战胜演术章）
特点：极短、极浓缩、每句话都包含核心哲理
```

### 1. 执行前检查

在开始之前，先确认：

```
✅ 原文已获取（来自维基文库/正统道藏）
✅ 辨明了是三章完整版
✅ 确认没有缺句错字
✅ 准备好了至少一种权威注本（刘处玄注或其他）做对照
```

### 2. 实体类别定义

《黄帝阴符经》的实体跟《史记》完全不同。没有人物官职，有的是**概念和原理**。

| 实体类别 | 含义 | 例子 |
|---------|------|------|
| **核心概念** | 书的根本概念 | 天道、五贼、盗机、三要 |
| **对偶概念** | 互相依存的矛盾对 | 天生/天杀、恩/害、生/死 |
| **修炼方法** | 具体的修行操作 | 伏藏、动静、三反昼夜 |
| **境界状态** | 修行的层次成果 | 圣人、神明、自然之道 |
| **类比意象** | 用来比喻的具象物 | 火木、龙蛇、瞽者聋者 |

### 3. 执行步骤

#### 第一步：逐章通读并做摘要（三章 × 5分钟）

对每章做一句话摘要：

```
第一章「神仙抱一演道章」：
  讲人要观察天道规律、按天道行事，核心在天人关系

第二章「富国安民演法章」：
  讲天地人三者互相"偷"（盗）的关系，核心在三才机制

第三章「强兵战胜演术章」：
  讲怎么修炼、怎么掌握阴阳变化的术，核心在修炼方法
```

#### 第二步：逐句标实体和关系（300字 × 逐句）

对**每一句**做标注，格式：

```
原文：观天之道，执天之行，尽矣。

实体识别：
  - 核心概念：天道（宇宙的根本规律）
  - 核心概念：天行（按规律做事）

关系提取：
  - 方法-目标：观 → 道（观察是为了理解道）
  - 方法-目标：执 → 行（执守是为了按道行动）
  - 总结-结论：尽矣（做到了就完了，无所欠缺）

原文解读：
  观察天道的运行规律，按天道的规律去执行，就够了。
  这是全书的开篇总纲——所有道理都在这两句里。
```

#### 第三步：提取对对偶概念对

《阴符经》大量使用对偶思维，要专门提取这些「一对一对」的概念：

```
天生 ↔ 天杀（道生万物，也杀万物）
恩 ↔ 害（恩情中藏着祸害，祸害中藏着恩情）
生 ↔ 死（生的尽头是死，死的尽头是生）
神 ↔ 不神（知道的神通，不如不知道的神通）
天地文理圣 ↔ 时物文理哲（外求天文 vs 内悟时物）
虞愚 ↔ 不愚圣（世人的愚昧 vs 修行者的不愚）
至乐 ↔ 至静（极致的快乐 vs 极致的宁静）
至私 ↔ 至公（天最自私，也最公平）
```

#### 第四步：结构化输出

输出格式：

```
# 《黄帝阴符经》知识结构化

## 一、核心体系

总纲：观天之道，执天之行，尽矣。
核心机制：五贼在心，施行于天 → 宇宙在乎手，万化生乎身
天人关系：天性，人也；人心，机也；立天之道，以定人也

## 二、三章要义

章1（神仙抱一演道）：
  - 论天人合一
  - 核心句：天人合发，万变定基
  - 大纲：观天→五贼→立人→杀机→伏藏→修炼

章2（富国安民演法）：
  - 论三才互盗
  - 核心句：三盗既宜，三才既安
  - 大纲：盗机→食时→神不神→日月→君子小人

章3（强兵战胜演术）：
  - 论修炼功法
  - 核心句：阴阳相推，而变化顺矣
  - 大纲：绝利一源→三反昼夜→恩害→自然之道→八卦甲子

## 三、核心概念词典

天道：宇宙的根本规律
天行：按天道行动
五贼：五行（金木水火土）的逆用
盗机：天地人互相取用的玄机
三要：九窍之中最关键的三个通道（耳目口）
伏藏：隐藏自己的智慧和机巧

## 四、修炼功法清单

1. 观天执天 → 理解天道
2. 伏藏巧拙 → 不露锋芒
3. 三要动静 → 控制耳目口的开关
4. 火生木、祸发必克 → 防祸于未然
5. 食其时、动其机 → 顺时而动
6. 绝利一源 → 专注一处
7. 三反昼夜 → 日夜反复修炼
8. 阴阳相推 → 顺应阴阳变化

## 五、对偶概念图谱

天生 ──↔── 天杀
恩 ──↔── 害
生 ──↔── 死
神 ──↔── 不神
至乐 ──↔── 至静
虞愚 ──↔── 不愚圣
至私 ──↔── 至公
└─ 核心逻辑：两者互相依存、互相转化 ──┘
```

### 4. 验收标准

```
✅ 300字全文逐句标注完成
✅ 核心概念词典至少提取10个概念
✅ 对对偶概念至少提取5对
✅ 修炼功法清单至少提取5条
✅ 结构化输出包含全部五个部分
✅ 至少参考一条权威注本（本示例参考刘处玄注）
```

### 5. 自更新备注

**如果发现以下问题，修改对应步骤：**
- 章节结构不同 → 修改第一步的章节划分
- 发现了新的实体类型 → 加到实体类别表中
- 参考的注本有争议 → 在解读中标注「争议」
