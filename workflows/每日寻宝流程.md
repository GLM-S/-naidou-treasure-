# 每日寻宝流程

## 时间
每天早上 7:30（cron: daily-oss-treasure）

## 步骤
1. 打开 kkgithub.com（GitHub国内镜像）
2. 查看 Trending 热门仓库
3. 打开 hf-mirror.com（HuggingFace国内镜像）
4. 查看热门模型
5. 筛选标准（优先级排序）：
   - 🥇 燃气险培训方向：交互式培训/H5工具/音频视频方案
   - 📖 书游时代方向：互动故事引擎/历史可视化/AI叙事
   - 🏛️ 知识工程：向量数据库/知识图谱/RAG方案
   - 🛠️ 通用：好用的AI工具/国内能部署的开源项目
6. 有价值的 → clone README到 treasure/
7. 特别好的 → 直接装到服务器
8. 更新 TREASURY.md 索引
9. 结果发给静渊

## 已掌握的宝藏来源
- kkgithub.com (GitHub镜像)
- hf-mirror.com (HuggingFace镜像)
- 静渊推荐（直接发给我）
