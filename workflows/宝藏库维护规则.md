# 宝藏库维护规则

## 架构
treasure/
├── TREASURY.md          ← 总索引（每次更新）
├── 寻宝记录-YYYY-MM-DD.md  ← 当日寻宝清单
├── README.md            ← 总说明
├── workflows/           ← 工作流程与方法论
├── skills-backup/       ← 所有技能（集群）
├── chat-history/        ← 所有聊天记录
├── outputs/             ← 所有文章产出
├── memory-backup/       ← 每日日志
├── scripts-backup/      ← 工具脚本
├── openclaw.json        ← 系统配置
├── [项目名]/            ← 感兴趣的开源项目（clone README）
└── [项目名]/            ← 已安装的工具（含配置备忘）

## 更新触发条件
- 每天早上 7:30 寻宝后 → 更新
- 安装了新工具 → 更新
- 写完了新文章 → 更新
- 改了什么重要配置 → 更新
- 静渊说"存起来" → 立刻存

## 标记规范
- ✅ 已装 — 能用
- 📌 感兴趣 — 先存着
- ❌ 不适用 — 评估后放弃
- ⭐ 推荐 — 值得优先看
